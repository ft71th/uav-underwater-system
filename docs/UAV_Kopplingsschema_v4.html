🟣 WAGO PLC — Funktioner som gateway

  • Vidarebefordra: PWM-data från boj → UAV Arduino för servo/ESC-styrning
  • Logga: GPS-position, trycksensordata, systemstatus
  • Styra ballast: Via DO → relä → linjärenhet (oberoende av Arduino)
  • Övervaka djup: Trycksensor 4-20mA → beräkna djup
  • Failsafe: Om boj-signal tappas → neutral PWM till Arduino, ballast upp
  • Framtid: DVL-data + IMU → autonom navigering

🟣 WAGO Modulordning på DIN-skena

  • PFC200 Controller → 750-652 #1 (Boj RS485) → 750-652 #2 (UAV RS485) → 750-466 (AI) → 750-530 (DO) → 750-626 (Matning) → 750-600 (Slutmodul)

📝 Viktiga noteringar

  • GEMENSAM JORD: Alla tre batterier (2×22.2V + 7.4V) MÅSTE dela GND-buss. Kritiskt för RS485 och I²C.
  • WAGO 24V-matning: Använd en regulerad DC/DC 22.2V→24V. WAGO tolererar 18–30V men vill ha stabil 24V.
  • Ballast nu via WAGO: WAGO styr linjärenheten direkt via DO → relä. PCA9685 CH4 frigjord.
  • Två separata RS485-bussar på WAGO: #1 (boj-kabel) och #2 (UAV Arduino) — ej samma buss!
  • WAGO som failsafe: Om boj-länk tappas kan WAGO skicka neutral till Arduino + ballast upp.
  • USB-programmering Arduino: Koppla bort MAX485 D0/D1 vid uppladdning.
  • RS485-terminering: 120Ω A↔B på kabeländar (boj-kabel). Intern WAGO→Arduino-länk är kort — behöver normalt ej terminering.
  • Avkoppling: 100µF + 100nF nära PCA9685 V+ skruvplint.
